## Installation

Install framework dependencies.
    ```bash  
    npm install
    ```

## Configuration

Create configuration files and fill them with data.
    ```bash
    cp example.env .env.development
    cp example.env .env.production
    ```

## Local Development

Run a local gatsby server.
    ```bash
    gatsby develop
    ```

## Deployment

Run a deploy script.
    ```bash
    npm run deploy
    ```
