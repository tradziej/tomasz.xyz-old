## Installation
    
1. Install global dependencies.
    ```bash
    npm install -g serverless
    ```
    
2. Install framework dependencies.
    ```bash  
    npm install
    ``` 

## Configuration

1. Create the configuration file.
    ```bash
    cp serverless.env.template.yml serverless.env.yml


## Local Development

1. Run a local server.
    ```bash
    npm run local
    ```
